Minimiza el DFA actual. 1º Creamos un conjunto de divisiones de estados. Le insertamos los estados finales y el resto. 2º Llamamos a create_new_partition para que cree nuevas particiones. 3º Paramos cuando veamos que la particion actual es igual a la anterior. 4º Llamamos a update para que nos actualize el DFA. 

 @param   vector_division Conjunto de divisiones actual.
 @param   vector_division_old Conjunto de divisiones anterior.
